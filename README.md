# Analytics Dashboard

A Java Spring Boot Full Stack application with Python utilities, built with Maven.

## Tech Stack

- Java 17+
- Spring Boot
- Maven 3.6+
- Python 3.8+

## Prerequisites 

- JDK 17 or higher
- Maven 3.6+
- Python 3.8+ with pip

## Project Structure
```
analytics-dashboard/
├── src/main/
│   ├── java/                  # Java source files                                    
│   └── resources/             # Application resources (config files, static files)
├── python-utils/              # Python utility scripts
├── pom.xml                    # Maven build file
└── README.md                  # Project documentation
```
